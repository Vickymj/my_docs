# Git Cheat Sheet
```bash
echo "# MyBot" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Vickymj/MyBot.git
git push -u origin main
```

```bash
git remote add origin https://github.com/Vickymj/MyBot.git
git branch -M main
git push -u origin main

```

This cheat sheet covers the most commonly used Git commands.

## Getting Started

```bash
git init
```
```bash
git clone <url>
```

## Prepare to Commit

```bash
git add <file>
git add --all
git add -A
git add .
git add *.txt
git add -p
git rm <file>
git reset <file>
git mv <old> <new>
git rm --cached <file>
git reset
git status
```

## Make Commits

```bash
git commit
git commit -m "message"
git commit -am "message"
```

## Branching

```bash
git switch <branch>
git checkout <branch>
git switch -c <branch>
git checkout -b <branch>
git branch
git branch --sort=-committerdate
git branch -d <branch>
git branch -D <branch>
```

## Diff Changes

```bash
git diff HEAD
git diff --staged
git diff
```

## Diff Between Commits

```bash
git show <commit>
git diff <commit1> <commit2>
git diff <commit> <file>
git diff <commit> --stat
git show <commit> --stat
```

## Discard Changes

```bash
git restore <file>
git checkout <file>
git restore --staged --worktree <file>
git checkout HEAD <file>
git reset --hard
git clean -f
git clean -fd
git stash
git stash pop
git stash list
```

## Edit History

```bash
git reset HEAD^
git commit --amend
git rebase -i HEAD~N
git reflog <branch>
git reset --hard <commit>
```

## Code Archaeology

```bash
git log
git log --graph
git log --oneline
git log <file>
git log --follow <file>
git log -G "search_text"
git blame <file>
```

## Restore Old Files

```bash
git checkout <commit> <file>
git restore <file> --source <commit>
```

## Remote Repositories

```bash
git remote add <name> <url>
git push origin main
git push
git push -u origin <branch>
git push --force-with-lease
git push --tags
git fetch origin main
git pull --rebase
git pull
git pull origin main
```

## Git Configuration

```bash
git config user.name "Your Name"
git config user.email "you@example.com"
git config --global user.name "Your Name"
git config --global user.email "you@example.com"
git config alias.st status
git config alias.co checkout
git config alias.br branch
git config alias.cm "commit -m"
man git-config
```

**Important Files:** `.git/config`, `~/.gitconfig`, `.gitignore
